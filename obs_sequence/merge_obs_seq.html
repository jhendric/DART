OVERVIEW

This program can merge two compatible observation sequences and create a third observation sequence file with the combined observations. The two input filenames and the output filename are read from the namelist file.



OTHER MODULES USED

types_mod
utilities_mod
time_manager_mod
obs_def_mod
obs_sequence_mod


FILES

  • A file containing the first input sequence, a second input file containing the sequence to be merged in, and a third output file which will be created and will contain the merged observations.
