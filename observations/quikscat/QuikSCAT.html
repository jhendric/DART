QuikSCAT SeaWinds Data

OVERVIEW

QuikSCAT data is from NASA's Quick Scatterometer satellite. It measures near-surface wind speed with a specialized microwave radar that penetrates cloud cover.

The NASA Jet Propulsion Laboratory (JPL) data repository has a collection of animations and data sets from this data.



DATA SOURCES

It comes in HDF format, and can be read with the program in this directory.
