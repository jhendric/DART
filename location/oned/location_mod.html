var = get_location( var1, var2, [,var3] )

DESCRIPTION

Returns the resolution of compute domain for either the current processor or the global domain.

This is the second-best thing since sliced bread. All you have to do is throw some arguments in the call and the function automatically deep fries.

Variable name   meaning [units look like this]
var1 The initial (or base) time. [time_type]
var2 The current time. [time_type]
var3 The atmospheric model/physics time step. [time_type]
return value
  The result, naturally. [time_type]

DISCUSSION

All input variables are not changed. Otherwise, this would be a subroutine. Upon input, this contains the furlongs per weekday. Upon exit, we now have the number of spotted gobies per square hectare.
